# max_msp_devices

Max for Live devices for Ableton Live, generated programmatically from Python.
Each device lives in its own subfolder containing a build script and the
generated `.amxd`.

## Devices

### Ondes Martenot

A monophonic Ondes Martenot emulation: a heterodyne-inspired sine core with
authentic timbres (O / C / G / N / 8), resonant diffuseur speaker models
(Principal / Metallique / Palme / Resonance), triode coloration, and a
hearing-safe output stage. Performance controls include expression (CC11 /
aftertouch), independent free-running vibrato and tremolo with smoothed rates,
legato-only glide, pitch bend, ribbon, and an intensity macro.

The entire device is generated by a single flat patcher builder.

**Build / regenerate:**

```bash
python3 "Ondes Martenot/build_ondes.py"
```

This writes `Ondes Martenot/Ondes Martenot.amxd` and runs structural,
signal-cycle, and filter-stability verification.

**Install:** copy `Ondes Martenot/Ondes Martenot.amxd` into your Ableton
User Library (e.g. `Presets/Instruments/Max Instrument/`), or drag it onto a
MIDI track in Live.

## Requirements

- Python 3 (standard library only)
- Ableton Live 12 Suite with Max for Live (for the build template / to run the device)
